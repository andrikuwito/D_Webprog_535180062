WWW

WorldWideWeb (kemudian berganti nama menjadi Nexus untuk menghindari kebingungan antara perangkat lunak dan World Wide Web) adalah browser web pertama [1] dan editor web pertama.[2] Program ini dihentikan pada tahun 1994. Program ini merupakan satu-satunya browser web yang ada[1] saat diluncurkan serta editor HTML WYSIWYG pertama. Kode sumber peramban ini dirilis ke domain publik pada 30 April 1993.[3][4] Beberapa bagian kode sumber program ini masih berada di Tim Berners-Lee's NeXTSTEP di museum CERN dan belum pulih karena status komputer sebagai artefak sejarah.Templat:Cititation needed Bertepatan dengan peringatan 20 tahun pusat penelitian meluncurkan web ke di dunia, sebuah proyek dexter dimulai di CERN untuk melestarikan perangkat keras dan perangkat lunak asli yang terkait dengan kelahiran Web.[5] WorldWideWeb adalah program pertama yang bukan saja menggunakan FTP (File Transfer Protocol) namun juga HTTP (Hypertext Transfer Protocol) yang dikembangkan Tim Berners-Lee pada 1989. WorldWideWeb dapat menampilkan berkas HTML tanpa gambar yang tersimpan.

Sejarah

Berners-Lee menulis apa yang kemudian dikenal sebagai WorldWideWeb pada NeXT Computer[4] selama kuarter ketiga tahun 1990, saat bekerja untuk CERN. Peramban yang pertama dapat diselesaikan pada 25 Desember 1990, setelah hanya dua bulan lamanya dikembangkan.[6] Pengembangan selanjutnya hanya beredar di antara rekan kerja Berners-Lee di CERN sebelum akhirnya dirilis ke publik, melalui newsgroup internet, pada Agustus 1991.[6] Pada saat ini, beberapa orang lain, termasuk Bernd Pollermann, Robert Cailliau, Jean-François Groff,[7] dan mahasiswa sarjana tamu Nicola Pellow - yang menulis line pada Mode Browser - terlibat dalam proyek tersebut.[6] Berners-Lee pada awalnya mengusulkan nama yang berbeda untuk aplikasi barunya, antara lain Tambang Informasi (The Mine of Information) dan Jaring Informasi (Information Mesh) sebagai proposal proposal. Pada akhirnya World Wide Web dipilih,[8] tetapi kemudian diubah menjadi Nexus untuk menghindari kebingungan antara World Wide Web dan browser web.[1] Tim ini sembuat "browser pasif" yang tidak memiliki kemampuan untuk mengedit kontennya karena sulit untuk mem-porting fitur ini dari sistem operasi NeXT ke sistem operasi lain. Melakukan Porting ke Sistem X Windows (X) tidak dimungkinkan karena tidak ada seorang pun dalam tim tersebut yang memiliki pengalaman dengan kode sumber X.[2] Berners-Lee dan Groff kemudian mengadaptasi banyak komponen WorldWideWeb ke dalam versi bahasa pemrograman C, serta menciptakan API libwww.[9] Sejumlah browser internet kemudian muncul, terutama ViolaWWW, yang diterbitkan oleh Mosaic untuk menjangkit popularitas tinggi, yang pada tahun 1993 telah menggantikan program WorldWideWeb. Para kru yang terlibat dalam program tersebut telah dipindahkan ke tugas lain, seperti menentukan standar dan pedoman untuk pengembangan World Wide Web (mis. HTML, dan berbagai protokol komunikasi). Pada 30 April 1993, direktur CERN merilis kode sumber WorldWideWeb ke dalam domain publik. Beberapa versi perangkat lunak tersebut masih tersedia online.[10] Berners-Lee pada awalnya mempertimbangkan untuk merilisnya di bawah Lisensi Publik Umum GNU, tetapi mereka akhirnya merilisnya ke domain publik.[11]
